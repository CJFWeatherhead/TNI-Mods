## About This Mod

Automatically added by the ModManager GUI when mods are installed via ModManagerGUI.ps1.

Its presence allows other mods to detect whether the mod collection was set up using the
mod manager or installed manually. No gameplay changes are made.

Other mods may declare this as an optional dependency to provide manager-aware behaviour,
such as displaying update guidance or manager-specific instructions.

## Details

- Added automatically by ModManagerGUI.ps1 during mod installation
- Acts as a version-tracked marker for the ModManager GUI installation method
- Safe to leave installed; has no side effects on gameplay
