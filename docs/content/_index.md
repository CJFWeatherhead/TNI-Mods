# Welcome to Tower Networking Inc Modding Kit

This repository is a comprehensive modding kit for the game [Tower Networking Inc](https://store.steampowered.com/app/2939600/Tower_Networking_Inc/). Create, customize, and manage mods to enhance your gaming experience!

## About This Project

Tower Networking Inc modding is powered by [Godot Sandbox](https://github.com/libriscv/godot-sandbox) using [libriscv](https://libriscv.no/). This modding kit enables you to:

- **Create Mods**: Develop mods in C/C++ or Lua (Lua is recommended for easier development)
- **Manage Mods**: Use our PowerShell-based Mod Manager for easy configuration
- **Share Mods**: Distribute your creations with the community

## What We Hope to Achieve

Our goal is to provide a robust, user-friendly modding ecosystem that:

1. **Empowers Creators**: Make it easy for anyone to create and share game modifications
2. **Simplifies Management**: Provide intuitive tools for installing and configuring mods
3. **Ensures Quality**: Maintain standards and documentation for all mods
4. **Builds Community**: Foster a collaborative environment for mod developers

## Getting Started

### For Players

- Browse our [Mod Collection](/mods/) to find mods that interest you
- Learn how to use the [Mod Manager](/mods/tools/modmanager/) to install and configure mods
- Download mods with a single click from each mod's page

### For Developers

- Check out the [GitHub repository](https://github.com/CJFWeatherhead/TNI-Mods) for development resources
- Read the mod creation guides in our README
- Contribute your own mods following our standards

## Key Features

### Lua Modding Support
Lua mods are the preferred method for modding due to their ease of development and transparency. All Lua mods are source-available.

### Mod Manager
Our PowerShell-based Mod Manager provides:
- Interactive text UI with color-coded menus
- Easy keyboard navigation
- Enable/disable mods with a click
- Configure mod parameters with validation

### Quality Standards
Every mod includes:
- Comprehensive metadata (using our schema)
- Detailed README documentation
- Configurable parameters for customization

## Browse Mods

Visit the [Mods section](/mods/) to explore available modifications, or check out the [Tools section](/mods/tools/) for utilities like the Mod Manager.
